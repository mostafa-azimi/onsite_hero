# ShipHero Onsite Compass Field Blueprint

## Required First-Pass Fields

- Account name
- Account ID
- Onsite location
- Visit objective
- Workflows observed
- Blockers or risks

## Recommended Account Context

- ARR
- Visit date
- Onsite lead
- Customer stakeholders
- Customer sentiment
- Follow-up owner
- Follow-up due date

## Operation Snapshot

- Facility profile
- Orders per day
- SKU count
- Shifts
- Integrations in scope
- Current pain points
- Training gaps or SOP drift

## Workflow Walkthrough

Capture status, severity, root cause, notes, and next step for:

- Receiving
- Putaway
- Inventory
- Replenishment
- Picking
- Packing
- Shipping
- Returns
- Integrations

## Root Cause Taxonomy

- Training
- SOP gap
- Configuration
- Integration
- Hardware or network
- Inventory accuracy
- Labor planning
- Product feedback

## Trend Tags

- Go-live readiness
- Training needed
- Workflow redesign
- Support follow-up
- Product feedback
- Integration risk
- ARR risk
- Expansion opportunity

## Closeout Outputs

- AI-ready summary
- Blockers and risks
- Recommendations
- Follow-up owner and date
- Exportable structured JSON
- Saved local report for trend review
